# Image Classification Project - UC13 (SENAC)

Image Classification Project developed as part of the UC13 (Curricular Unit 13) coursework at SENAC.

## Description

This project aims to build a Deep Learning model capable of classifying images.  
The model was trained to recognize visual patterns in images and classify them into different categories.

The application demonstrates the full pipeline of an AI project:
- Image processing
- Model training
- Performance evaluation

## Technologies Used

- Python
- PyTorch
- Pandas
- Matplotlib
- Jupyter Notebook

# Projeto de Classificação de Imagens - UC13 (SENAC)

Projeto de Classificação de Imagens desenvolvido como atividade da Unidade Curricular 13 (UC13) do SENAC.

## Descrição do Projeto

Este projeto tem como objetivo construir um modelo de Deep Learning capaz de classificar imagens.  
O modelo foi treinado para reconhecer padrões visuais em imagens e classificá-las em diferentes categorias.

A aplicação demonstra todo o pipeline de um projeto de IA:
- Processamento de imagens
- Treinamento do modelo
- Avaliação de desempenho

## Tecnologias Utilizadas

- Python
- Pytorch
- Pandas
- Matplotlib
- Jupyter Notebook
